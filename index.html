Web Audo API Code Examples

A growing list of code examples for you to fiddle with and get started using the Web Audio API

Example 1: Creating an Oscillator

This example demonstrates creating a sine wave oscillator. Sine wave is the default wave type but square, triangle and sawtooth wave types are also available.

Example 2: Controlling the Frequency of an Oscillator

Here we update the oscillators frequency using the current X position of the mouse. The effect resembles that of a Theremin; one of the earliest electronic instruments.

Example 3: Controlling the Output Gain

Now we have a range input to control the output gain. Gain nodes accept a value between 0 and 1

Example 4: Creating a Filter

Many types of filters can be created using the createBiquadFilter method. This examples uses a "lowself" filter controlled by the mouses y position
